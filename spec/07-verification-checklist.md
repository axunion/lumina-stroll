# 07. 最終手動 QA チェックリスト

`pnpm build && pnpm preview` の本番ビルドで実施する（出荷チェックマイルストーン — `05-roadmap.md`）。全項目にチェックが付いたら完成とみなす。§1〜§10 は Phase 1 の項目（実施済み — M13 では §15 の回帰として再実施する）、§11〜§15 は Phase 2 の項目。

## 1. 起動と表示

- [ ] エラー・警告なしでページが表示される（DevTools Console が空。ただし **未配置スプライトの 404 ネットワークログは許容** — `01-architecture.md` §8）
- [ ] 暗い世界の中央付近にプレイヤーの光る円が表示され、周囲だけ柔らかく見える
- [ ] HUD に `0 / 14`（Sparkles）、`0 / 6`（Flame）、`Enchanted Forest` が表示される
- [ ] タブタイトルが "Lumina Stroll"

## 2. 移動と入力

- [ ] 矢印キーで 4 方向 + 斜めに移動できる
- [ ] WASD でも同様に移動できる
- [ ] 斜め移動が縦横移動より速くない
- [ ] 矢印キーを押してもページがスクロールしない
- [ ] Cmd/Ctrl を含むショートカット（例: Cmd+R）が普通に効く
- [ ] ワールドの四辺で止まる（跳ね返り・すり抜けなし）
- [ ] 移動中だけ足跡パーティクルが残り、停止すると出ない
- [ ] 別タブに切り替えて数秒後に戻っても、プレイヤーがワープしていない

## 3. リサイズ・DPR

- [ ] ウィンドウをリサイズしても描画が崩れず、canvas がぼやけない
- [ ] ブラウザズーム（Cmd +/−）後も crisp に描画される
- [ ] リサイズ後もライティング（暗幕の穴）の位置がプレイヤーとずれない

## 4. クリスタル

- [ ] クリスタルが上下にボブしており、個体ごとにタイミングがずれている
- [ ] 触れるとバースト + "+1" テキストが出て、HUD が +1 される
- [ ] 同じ場所を往復しても同じクリスタルで二重加算・復活がない
- [ ] 14 個すべて取得すると HUD が `14 / 14` になる

## 5. ブレイジャー

- [ ] 未点灯時は暗い残り火として、かすかに視認できる
- [ ] 近づくだけで自動点灯し、青紫の炎が上がる（ボタン操作不要）
- [ ] 点灯は約 0.6 秒かけて光が広がる（瞬間切替ではない）
- [ ] 離れて戻っても点いたままで、その一帯が明るい
- [ ] 6 個すべて点灯すると HUD が `6 / 6` になる
- [ ] 境界（x≈2450）のランドマークブレイジャーが目印として機能している

## 6. バイオーム

- [ ] x=2400 付近を歩くと背景・タイル・環境パーティクルの色が滑らかに変わる（ハードスワップなし）
- [ ] 境界を越えると HUD のバイオーム名が `Crystal Cave` に切り替わる（1 回だけ、チラつきなし）
- [ ] 境界を往復してもクラッシュ・色の跳ねがない

## 7. メニュー（Dialog / Tabs）

- [ ] Settings ボタンでメニューが開き、背景の移動が止まる
- [ ] フォーカスがダイアログ内にトラップされる（Tab 巡回がダイアログ外へ出ない）
- [ ] Escape で閉じ、フォーカスが Settings ボタンに戻る
- [ ] 移動キーを押したまま Escape で閉じても、勝手に歩き続けない（stuck key なし）
- [ ] Journal タブの集計（クリスタル・ブレイジャー・現在バイオーム）が HUD と一致する
- [ ] Tabs を矢印キーで切り替えられる（Kobalte 既定挙動）

## 8. reduced-motion

`02-game-design.md` §9 のマトリクスに従い、Settings のトグルを ON にして確認:

- [ ] 足跡パーティクルが出なくなる
- [ ] クリスタル取得がリングストローク 1 本になる（"+1" はその場フェード）
- [ ] 環境パーティクルが消える
- [ ] クリスタルのボブが止まる
- [ ] 炎が定常グローになる（揺らぎ・炎パーティクルなし）
- [ ] プレイヤー光のパルスが止まる
- [ ] バイオームの色 lerp は **効いたまま**
- [ ] Dialog の出入りアニメーションがなくなる
- [ ] トグルを OFF に戻すとすべて復帰する
- [ ] DevTools Rendering → `prefers-reduced-motion: reduce` をエミュレートしてリロードすると、トグルが最初から ON

## 9. モバイル（DevTools タッチエミュレーション）

- [ ] D-pad が右下に表示される（デスクトップでは非表示）
- [ ] 各ボタンを押している間だけ対応方向に移動する
- [ ] ボタンを押したまま指をずらして離しても移動が止まる（pointerleave / pointercancel）
- [ ] D-pad 操作でページがスクロール・ズームしない

## 10. 安定性・クロスブラウザ

- [ ] 5 分放置ソーク: パーティクル総数が `CONFIG.maxParticles` を超えず、メモリ・FPS が安定している（DevTools Performance で確認）
- [ ] 全マップ踏破プレイスルー: 端から端まで歩き、全取得・全点灯・両バイオーム到達までクラッシュなし
- [ ] Chrome で全項目確認
- [ ] Safari で主要項目（起動・移動・ライティング・メニュー・取得）を確認

## 11. スプライト差し替え

`03-reference.md` §7 の 9 キーで確認（検品用の手描きダミー PNG で可）:

- [ ] 素材ゼロの状態で、従来どおりの手続き描画で全機能が動く
- [ ] `player.png` を置いてリロードすると差し替わる。消してリロードすると手続き描画に戻る（ビルド・設定変更なし）
- [ ] 一部のキーだけ置いた状態でも正常（置いたものだけ差し替わる）
- [ ] タイルは片方だけなら市松のまま。両方置くと全面テクスチャになり、ブレンド帯（x≈2200〜2600）で 2 種が滑らかにクロスフェードする
- [ ] player / crystal / 点灯ブレイジャーのスプライトの背後にグローが残っている（ハロー維持則）
- [ ] クリスタルのボブ・点灯イーズ・フリッカーがスプライトでも従来どおり効く
- [ ] リサイズ・ブラウザズーム後もスプライトが crisp（にじまない）
- [ ] 暗幕・光の穴の位置がスプライトとずれない。reduced-motion の各行が素材ありでも表のとおり
- [ ] ファイル名を意図的に間違えると、エラーなくフォールバックし、Network タブに 404 が見える

## 12. 進捗セーブ

- [ ] クリスタル取得・ブレイジャー点灯・碑文発見 → リロードで保持される（HUD・Journal・ワールドすべて一致）
- [ ] 復元されたブレイジャーは起動直後から点灯済みで、点灯アニメーションを再生しない
- [ ] プレイヤーはリロード後、常に入口（スポーン地点）から始まる
- [ ] DevTools で `luminaStroll.save` の JSON を破壊 → リロードでクラッシュ・エラー表示なく新規開始
- [ ] リセットボタン: 1 回目で armed（文言変化 + 約 4 秒で自動復帰）、2 回目で初期状態に戻る
- [ ] Safari プライベートモード（または localStorage 無効化）でもクラッシュせず遊べる（保存されないだけ）

## 13. サウンド

- [ ] ページロード直後は完全に無音。Console に autoplay 警告が出ない
- [ ] 最初のキー入力またはタップでアンビエントが静かに始まる
- [ ] バイオーム境界を往復すると、色と同じ帯域で音が滑らかにクロスフェードする（ステップ変化・プツ音なし）
- [ ] 取得チャイム・点灯スウェル・碑文ベルがそれぞれのイベントで 1 回鳴る
- [ ] Sound トグル OFF で即座に無音になり、リロード後も OFF が保持される
- [ ] メニューを開いてもアンビエントは途切れない（ワンショットは発生しない）
- [ ] reduced-motion ON でも音は変わらない（独立）

## 14. 小さな発見

- [ ] ウィスプが漂っており、近づくとゆるやかに寄ってきて、離れると徘徊に戻る（急加速しない）
- [ ] 光る草が近接で明るくなり、離れても微光が残る（完全消灯しない）
- [ ] 石碑に近づくと詩文が表示され、圏外に出て再訪すると再表示される
- [ ] 発見（Journal 追加・セーブ・ベル）は各碑文で初回のみ
- [ ] Journal の碑文欄: 発見済みは詩文、未発見は "…" のロック行、集計が `n / 6`
- [ ] reduced-motion ON: ウィスプが消え、草のグローは効いたまま、碑文テキストはその場フェード
