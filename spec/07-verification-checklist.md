# 07. 最終手動 QA チェックリスト

`pnpm build && pnpm preview` の本番ビルドで実施する（M7 — `05-roadmap.md`）。全項目にチェックが付いたら完成とみなす。

## 1. 起動と表示

- [ ] エラー・警告なしでページが表示される（DevTools Console が空）
- [ ] 暗い世界の中央付近にプレイヤーの光る円が表示され、周囲だけ柔らかく見える
- [ ] HUD に `0 / 14`（Sparkles）、`0 / 6`（Flame）、`Enchanted Forest` が表示される
- [ ] タブタイトルが "Lumina Stroll"

## 2. 移動と入力

- [ ] 矢印キーで 4 方向 + 斜めに移動できる
- [ ] WASD でも同様に移動できる
- [ ] 斜め移動が縦横移動より速くない
- [ ] 矢印キーを押してもページがスクロールしない
- [ ] Cmd/Ctrl を含むショートカット（例: Cmd+R）が普通に効く
- [ ] ワールドの四辺で止まる（跳ね返り・すり抜けなし）
- [ ] 移動中だけ足跡パーティクルが残り、停止すると出ない
- [ ] 別タブに切り替えて数秒後に戻っても、プレイヤーがワープしていない

## 3. リサイズ・DPR

- [ ] ウィンドウをリサイズしても描画が崩れず、canvas がぼやけない
- [ ] ブラウザズーム（Cmd +/−）後も crisp に描画される
- [ ] リサイズ後もライティング（暗幕の穴）の位置がプレイヤーとずれない

## 4. クリスタル

- [ ] クリスタルが上下にボブしており、個体ごとにタイミングがずれている
- [ ] 触れるとバースト + "+1" テキストが出て、HUD が +1 される
- [ ] 同じ場所を往復しても同じクリスタルで二重加算・復活がない
- [ ] 14 個すべて取得すると HUD が `14 / 14` になる

## 5. ブレイジャー

- [ ] 未点灯時は暗い残り火として、かすかに視認できる
- [ ] 近づくだけで自動点灯し、青紫の炎が上がる（ボタン操作不要）
- [ ] 点灯は約 0.6 秒かけて光が広がる（瞬間切替ではない）
- [ ] 離れて戻っても点いたままで、その一帯が明るい
- [ ] 6 個すべて点灯すると HUD が `6 / 6` になる
- [ ] 境界（x≈2450）のランドマークブレイジャーが目印として機能している

## 6. バイオーム

- [ ] x=2400 付近を歩くと背景・タイル・環境パーティクルの色が滑らかに変わる（ハードスワップなし）
- [ ] 境界を越えると HUD のバイオーム名が `Crystal Cave` に切り替わる（1 回だけ、チラつきなし）
- [ ] 境界を往復してもクラッシュ・色の跳ねがない

## 7. メニュー（Dialog / Tabs）

- [ ] Settings ボタンでメニューが開き、背景の移動が止まる
- [ ] フォーカスがダイアログ内にトラップされる（Tab 巡回がダイアログ外へ出ない）
- [ ] Escape で閉じ、フォーカスが Settings ボタンに戻る
- [ ] 移動キーを押したまま Escape で閉じても、勝手に歩き続けない（stuck key なし）
- [ ] Journal タブの集計（クリスタル・ブレイジャー・現在バイオーム）が HUD と一致する
- [ ] Tabs を矢印キーで切り替えられる（Kobalte 既定挙動）

## 8. reduced-motion

`02-game-design.md` §9 のマトリクスに従い、Settings のトグルを ON にして確認:

- [ ] 足跡パーティクルが出なくなる
- [ ] クリスタル取得がリングストローク 1 本になる（"+1" はその場フェード）
- [ ] 環境パーティクルが消える
- [ ] クリスタルのボブが止まる
- [ ] 炎が定常グローになる（揺らぎ・炎パーティクルなし）
- [ ] プレイヤー光のパルスが止まる
- [ ] バイオームの色 lerp は **効いたまま**
- [ ] Dialog の出入りアニメーションがなくなる
- [ ] トグルを OFF に戻すとすべて復帰する
- [ ] DevTools Rendering → `prefers-reduced-motion: reduce` をエミュレートしてリロードすると、トグルが最初から ON

## 9. モバイル（DevTools タッチエミュレーション）

- [ ] D-pad が右下に表示される（デスクトップでは非表示）
- [ ] 各ボタンを押している間だけ対応方向に移動する
- [ ] ボタンを押したまま指をずらして離しても移動が止まる（pointerleave / pointercancel）
- [ ] D-pad 操作でページがスクロール・ズームしない

## 10. 安定性・クロスブラウザ

- [ ] 5 分放置ソーク: パーティクル総数が `CONFIG.maxParticles` を超えず、メモリ・FPS が安定している（DevTools Performance で確認）
- [ ] 全マップ踏破プレイスルー: 端から端まで歩き、全取得・全点灯・両バイオーム到達までクラッシュなし
- [ ] Chrome で全項目確認
- [ ] Safari で主要項目（起動・移動・ライティング・メニュー・取得）を確認
